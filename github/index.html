Gitとは？

Git（ギット）はバージョン管理システムの一つ

　

バージョン管理システムとは

ファイルの作業のタイミング毎にバージョンを記録し、いつでも記録した時点に戻すことができるシステム

　

バージョン管理システムの種類

・Git(ギット)　←　手軽に使えるクライアントソフトが豊富

・CVS(カレントバージョンシステム)

・Subversion(サブバージョン)

・Mercurial(マーキュリアル)

Sourcetreeのインストール

Sourcetreeとは

　Gitを使うためのクライアントソフト

　

インストール

　https://www.sourcetreeapp.com/

　ライセンス同意 → I agree～ をクリック

　BitbucketCloud → スキップ

　Gitにチェックを入れる

　名前(GTA24_DenpaTarou)とメールアドレス(nkc2024****@st.denpa.jp)を入力

　SSHキーの読み込み → いいえ

　

　※ 起動したSourcetreeを終了しておく

Visual Studio Codeのインストール

Visual Studio Codeとは

　コードやテキストを編集するエディターソフト

　

インストール

　https://code.visualstudio.com/

　【次へ】で進めていき、最後に【インストール】

日本語化

　Extensions（画面左側メニューの一番下のアイコン）のボタンをくりっくし、検索ウィンドウで「japanese」と入力

　【Japanese Language Pack】をクリックし、【Install】をクリック

　【Restart】をクリック

　

　※ 配色等の変更は【ファイル】【ユーザー設定】【配色テーマ】

Git用語

リポジトリ（Repository）

　「倉庫」といった意味の英単語でファイルを管理しておく場所

　ローカルリポジトリ・・・自分のPC上にあるリポジトリ

　リモートリポジトリ・・・ネット上にあるリポジトリ

　

ワークスペース（Workspace）

　Gitでは単なるフォルダの事を指す。

　リポジトリを作成したときに指定するフォルダの事

　

コミット（Commit）

　「委託」といいった意味の英単語で、リポジトリにファイルを預けること

　

インデックス（Index）

　「目次」といった意味の英単語で、リポジトリに引き渡すファイルの一覧のこと

　

ステージング（Staging）

　「リハーサル会場」的な意味合いで、変更したファイルをインデックスに追加することを指す

　

リセット（Reset）

　作業を無かったことにする

　

ブランチ（Branch）

　「枝」という意味の英単語で、１つのファイルやプロジェクトを複数の「バージョン」に分けることを指す
