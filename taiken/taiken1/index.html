本を買わないプログラマは成長しない。

新しいことにチャレンジしないプログラマはいらない。

在学中に行うべきことは、【自分で調べて問題を解決できる能力】を養うこと。

どんなことにも興味を持ち、実践してみる。

世の中のクリエイターは皆、そうやって生きています。

Unityを起動してみよう

「Unity Hub」をダブルクリックして新規プロジェクトを作成

テンプレート「２D」

プロジェクト名「Taiken」

保存先「デスクトップ」

「NKCUFO2D.unitypackage」をインポート

完成版で遊んでみよう

Assets - Scenes - Complete をダブルクリック　→　再生

Enterキーでスタート

カーソルキーで移動

制限時間内にアイテムをすべて取ればクリアー

背景を表示しよう

Assets - Scenes - Taiken をダブルクリック

Assets - Sprites - Backgroundをヒエラルキーにドラッグ＆ドロップ

設定

　Sorting Layer:Background

　コンポーネントを追加（BoxCollider2Dを４つ追加）

　BoxCollider2D_01:左 Offset(-14.3, 0) Size(3.3, 31.64)

　BoxCollider2D_02:右 Offset( 14.3, 0) Size(3.3, 31.64)

　BoxCollider2D_03:上 Offset(0, 14.3) Size(31.64, 3.3)

　BoxCollider2D_04:下 Offset(0, -14.3) Size(31.64, 3.3)

プレーヤーを作ろう

Assets - Sprites - Playerをヒエラルキーにドラッグ＆ドロップ

設定

　Sorting Layer:Player

　

Rigidbody2Dコンポーネントを追加

　Gravity Scale 0

　

Assets - Scripts - PlayerController.csをアタッチ

PlayerController.csに以下の命令を追加


	// ここに命令を追加↓以下を追加
　　rb2d.AddForce(moveVec * speed);

　

Main Camera に Scripts - CameraController.cs を追加

当たり判定を追加しよう

PlayerにCircle Collider 2Dコンポーネントを追加

　Radius 2.15

　

※ 壁と当たるか確認する

アイテムをつくろう

Assets - Sprites - Pickup をヒエラルキーにドラッグ＆ドロップ

設定

　Sorting Layer:Pickups

　Tag - PickUp

　

Circle Collider 2Dコンポーネントを追加

　Radius 0.84

　Is Trigger ON

Assets - Scripts - Rotator.csをアタッチ

　

　※ 実行してプレーヤーが当たったら消えるか確認し、その後コピーして１２個配置する

UIの表示

Prefabs から Canvas をヒエラルキーに追加

Prefabs から SceneDirector をヒエラルキーに追加

　

※ Enterキーでスタート

※ 時間が余ったらアイテムの配置を考えて難易度を調整しよう
