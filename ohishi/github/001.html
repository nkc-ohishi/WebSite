Gitとは？

Git（ギット）はバージョン管理システムの一つ

　

バージョン管理システムとは

ファイルの作業のタイミング毎にバージョンを記録し、いつでも記録した時点に戻すことができるシステム

　

バージョン管理システムの種類

・Git(ギット)　←　手軽に使えるクライアントソフトが豊富

・CVS(カレントバージョンシステム)

・Subversion(サブバージョン)

・Mercurial(マーキュリアル)

Visual Studio Codeのインストール

Visual Studio Codeとは

　コードやテキストを編集するエディターソフト

　

インストール

　https://code.visualstudio.com/

　【次へ】で進めていき、最後に【インストール】

日本語化

　Extensions（画面左側メニューの一番下のアイコン）のボタンをクリック

　検索ウィンドウで「japanese」と入力

　【Japanese Language Pack】をクリックし、【Install】をクリック

　【Restart】をクリック

Git用語

リポジトリ（Repository）

　「倉庫」といった意味の英単語でファイルを管理しておく場所

　ローカルリポジトリ・・・自分のPC上にあるリポジトリ

　リモートリポジトリ・・・ネット上にあるリポジトリ

　

ワークスペース（Workspace）

　Gitでは単なるフォルダの事を指す。

　リポジトリを作成したときに指定するフォルダの事

　

コミット（Commit）

　「委託」といいった意味の英単語で、リポジトリにファイルを預けること

　

インデックス（Index）

　「目次」といった意味の英単語で、リポジトリに引き渡すファイルの一覧のこと

　

ステージング（Staging）

　「リハーサル会場」的な意味合いで、変更したファイルをインデックスに追加することを指す

　

リセット（Reset）

　作業を無かったことにする

　

ブランチ（Branch）

　「枝」という意味の英単語で、１つのファイルやプロジェクトを複数の「バージョン」に分けることを指す

　

マージ（Merge）

　「合併」という意味の英単語で、枝分かれしたブランチを1本に合流させることを指す

　

リベース（Rebase）

　「再び土台に戻す」といった意味で、ブランチを取り込むことを指す

　

タグ（Tag）

　履歴につける目印

　

パッチ（Patch）

　変更部分だけを抜き取った部分的なソース

　

プル（Pull）

　「引っ張る」という意味の英単語で、リモートリポジトリからローカルリポジトリにファイルを引き出すことを指す

　

フェッチ（Fetch）

　「取りに行く」という意味の英単語で、変更箇所を検出するときに使う

　

スタッシュ（Stash）

　「隠す」という意味の英単語で、ファイルへの変更を一時的に保存するときに使う

　

スカッシュ（Squash）

　「押しつぶす」という意味の英単語で、複数のコミットを1個にまとめる作業を指す
